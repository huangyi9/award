# 微信小程序转盘抽奖
----

## 简介

用微信小程序开发的Canvas绘制可配置的转盘抽奖。      

使用[https://github.com/givebest/GB-canvas-turntable](http://blog.givebest.cn/GB-canvas-turntable.html)代码移植过而来。

      
## 说明

1.布局使用大量绝对定位，存在不同设备视图兼容问题
2.所有文件路径必须避免中文字符
3.未适配ipad
4.字体不可用
